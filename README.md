# Synapse
Discord bot
Website: https://synapsebot.netlify.app/

## Commands
##### Basic Data
X Stock (Ticker)
##### Advanced Data
X AdvancedData (Ticker)
##### Graph
X Graph (Ticker)
##### Custom Graph
X CGraph (days) (Ticker)
##### Company Data
X Company (Ticker)
##### Crypto Info
X Crypto (Ticker)
##### Return
X Return (Ticker)
##### Unformatted Data
X Data (Ticker)
##### Weather
X Weather (City)

## Sim Commands
##### Start Game (10k)
X Start
##### Buy Stock
X Buy (Ticker) (Amount)
##### Sell Stock
X Sell (Ticker) (Amount)
##### Portfolio
X Portfolio
